# Ai-voice-assistant
An AI-powered voice assistant built using Python that can take voice or text input, answer questions using a local LLM, speak responses aloud, open applications, and perform web searches.
# AI Voice Assistant

This project is an AI-powered voice assistant built using Python.

## Features
- Voice input using microphone
- AI responses using Ollama
- Text-to-speech output
- Open applications
- Web search functionality

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Ollama

## How to Run

1. Install dependencies
pip install speechrecognition pyttsx3 requests

2. Start Ollama
ollama serve

3. Run the assistant
python assistant.py
